# CAA_16S-analysis

Tidewater Glaciers Analysis

This repository contains R code and metadata required for analyzing the publication by Spence, J.S., et al (submitted 2026) "Glacier-induced upwelling shapes microbial communities in Arctic marine systems". The code processes and visualizes the plots referenced in the paper. All data required for creating plots is also found in the repository under data. Data was split into the upper 50 m of the water column, and entire water column, as observed in paper figures.
